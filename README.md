# 🎮 Rock Paper Scissors Game

A simple, modern, and animated **Rock Paper Scissors** web game built using **HTML, CSS, and JavaScript**.

---

## 🔗 Live Demo

👉 https://saimali35.github.io/Rock-Paper-Scissor-game/

---

## 📌 Overview

This project is a browser-based Rock Paper Scissors game where the player competes against the computer. The game includes smooth animations, responsive design, score tracking, and an interactive result modal.

It is a beginner-friendly frontend project created to practice JavaScript DOM manipulation, event handling, animations, and responsive UI design.

---

## ✨ Features

- 🎨 Clean and modern UI
- ⚡ Smooth animations and hover effects
- 🧠 Random computer move generation
- 📊 Real-time score tracking
- 🪟 Animated result modal
- 🔄 Play Again functionality
- ♻️ Reset Game button
- 📱 Responsive layout
- 🚀 Hosted using GitHub Pages

---

## 🕹️ How to Play

1. Open the game in your browser.
2. Click one of the options:
   - Rock
   - Paper
   - Scissors
3. The computer randomly selects its move.
4. The winner is displayed in the result modal.
5. Continue playing using **Play Again**.
6. Use **Reset Game** to reset the scores.

## 🚀 Run Locally

### Option 1: Open Directly

Simply open:

```bash
index.html
```

in your browser.

---

### Option 2: Using Python Local Server

Run the following command inside the project folder:

```bash
python -m http.server 8000
```

Then open:

```bash
http://localhost:8000
```
