# Rock Paper Scissor

A simple, animated Rock Paper Scissors web game built with HTML, CSS and JavaScript.

Live demo: https://saimali35.github.io/Rock-Paper-Scissor-game/

## Overview

This project is a small browser game where you choose Rock, Paper, or Scissors and play against the computer. The UI includes an animated result modal, score tracking, and responsive hover/interaction effects.

## Features

- Clean, modern UI with animations and visual feedback
- On-screen result modal showing both choices
- Scoreboard for player and computer
- Reset and Play Again controls
- Easy to run locally or host on GitHub Pages

## How to Play

1. Open the game in a browser.
2. Click one of the three icons (Rock, Paper, Scissors).
3. The result modal will show both choices and the round result.
4. Use `Play Again` to close the modal and continue, or `Reset Game` to reset scores.

## Files

- `index.html` — Game markup
- `style.css` — Styling and animations
- `script.js` — Game logic
- `images/` — Rock, Paper, Scissors assets

## Run Locally

Recommended: serve the folder with a simple local server to avoid any browser restrictions.

Using Python 3 (from the project root):

```bash
python -m http.server 8000
# then visit http://localhost:8000
```

Or simply open `index.html` in your browser for a quick test.

## Deployment

This repository is configured for GitHub Pages. The site is published at:

https://saimali35.github.io/Rock-Paper-Scissor-game/

If the site does not appear immediately, open the repository Settings → Pages and ensure the `gh-pages` branch is selected as the source.
